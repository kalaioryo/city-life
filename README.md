## Iniziare
<hr>

Per ottenere una copia locale 

### Prerequisiti

  - npm

    ```
    npm i latest-version
    ```
### Installazione

  1. Clona il repository
    
    ```
    gh repo clone kalaioryo/city-life
    ```
  2. installa i pacchetti NPM
    ```
    npm install
    ```
  3. Build dalla fonte
    ```
    npm run build
    ```
